❔ Помощь и инструкция

📌 Главный экран

• Крупные карточки: Зарплата, Доходы, Расходы, Остаток.
• Блок «Быстрое действие» — кнопка «➕ Новая операция».
• Последние 5 операций — клик для редактирования, крестик — удаление.
• Кнопка «Очистить все» — удаляет все операции.

📋 Операции

• Поиск по тексту, дате, типу.
• Клик по операции — редактирование.

🗂️ Категории

• Клик по названию — переименовать.
• Кнопки «+ Подкат. доход/расход» — добавить подкатегорию.
• Клик по подкатегории — редактировать, крестик — удалить.

🧮 Инструменты

• Калькулятор и конвертер валют с историей.
• История конвертации — полный просмотр и удаление записей.

📓 Блокнот

• Создавайте, редактируйте, удаляйте страницы.

📊 Данные (Статистика)

• Выберите период: день, неделя или месяц — диаграмма покажет соотношение доходов/расходов за выбранный промежуток.
• Кнопка «Сбросить статистику» — обнуляет накопленные данные и начинает отсчёт только с текущего момента (старые операции игнорируются).
• Анимация круга и процентов запускается при каждом обновлении.

Все данные сохраняются в браузере автоматически.
