# ШАГ 2 — УСТАНОВКА NODE.JS

Node.js — программа которая запускает JavaScript на компьютере. Без неё ничего не работает.

В терминале введите команды по одной (ждите завершения каждой):

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
```
*(Введите пароль когда спросит — звёздочки не видны, это нормально)*

```bash
sudo apt-get install -y nodejs
```

Проверьте что установилось:
```bash
node --version
npm --version
```
Должно написать что-то вроде: `v20.11.0` и `10.2.4`

### 9.1 — Разрешения для интернета

Слева в дереве файлов найдите и откройте:
`app` → `manifests` → `AndroidManifest.xml`

Найдите строку которая начинается с `<manifest`:
```xml
<manifest xmlns:android=...
```

СРАЗУ ПОСЛЕ этой строки (перед строкой `<application`) вставьте:
```xml
    <uses-permission android:name="android.permission.INTERNET" />
    <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />
    <uses-permission android:name="android.permission.VIBRATE" />
    <uses-permission android:name="android.permission.USE_BIOMETRIC" />
    <uses-permission android:name="android.permission.USE_FINGERPRINT" />
    <uses-permission android:name="android.permission.RECEIVE_BOOT_COMPLETED" />
```

Нажмите **Ctrl+S** для сохранения.

### 9.2 — Цвет темы

Слева найдите: `app` → `res` → `values` → `styles.xml`

Откройте и найдите строки с `colorPrimary`. Замените цвета:

# ШАГ 11 — СБОРКА APK ФАЙЛА

### Debug APK (для себя и тестирования)

В Android Studio:
1. Меню **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
2. Ждите 2-3 минуты
3. Внизу появится: "APK(s) generated" — нажмите **locate**

Этот файл можно скинуть на телефон и установить!

### Release APK (для Google Play)

1. **Build** → **Generate Signed Bundle / APK**
2. Выберите **APK** → Next
3. Нажмите **Create new** (создать ключ подписи):
   - Key store path: `/home/ваше_имя/BudgetPRO/keystore.jks`
   - Password: придумайте пароль (ЗАПИШИТЕ! без него нельзя обновить приложение)
   - Alias: `budgetpro`
   - Key password: тот же пароль
   - First and Last Name: ваше имя
   - Country Code: GE (для Грузии)
4. OK → Next → выберите **release** → Finish

APK будет в: `android/app/release/app-release.apk`

### 12.4 — Код для push-уведомлений

Добавьте в начало вашего index.js (после всех let/const деклараций):

```javascript
// ── PUSH УВЕДОМЛЕНИЯ (только в нативном приложении) ──
async function initPushNotifications() {
  // Проверяем что мы в Capacitor (нативное приложение)
  if (!window.Capacitor) return;
  
  try {
    const { PushNotifications } = await import('@capacitor/push-notifications');
    
    // Запрашиваем разрешение
    const permission = await PushNotifications.requestPermissions();
    if (permission.receive !== 'granted') return;
    
    // Регистрируемся
    await PushNotifications.register();
    
    // Получаем токен (отправьте его себе чтобы знать как отправить уведомление)
    PushNotifications.addListener('registration', (token) => {
      console.log('Push token:', token.value);
      localStorage.setItem('pushToken', token.value);
      // Показываем токен в настройках для отладки
    });
    
    // Получаем уведомление когда приложение открыто
    PushNotifications.addListener('pushNotificationReceived', (notification) => {
      showToast('🔔 ' + notification.title + ': ' + notification.body, 'info');
    });
    
    // Клик по уведомлению когда приложение закрыто
    PushNotifications.addListener('pushNotificationActionPerformed', (action) => {
      const data = action.notification.data;
      if (data.tab) setTab(data.tab);
    });
    
  } catch(e) {
    console.log('Push не доступен:', e);
  }
}

// Вызываем при старте
document.addEventListener('DOMContentLoaded', () => {
  setTimeout(initPushNotifications, 2000);
});
```

# ШАГ 13 — ОБЩЕНИЕ С ПОЛЬЗОВАТЕЛЯМИ (TELEGRAM BOT)

Telegram Bot — лучший способ общения создателя с пользователями.

### 13.1 — Создание бота

1. Откройте Telegram
2. Найдите **@BotFather**
3. Отправьте `/newbot`
4. Введите имя бота: **БюджетPRO Поддержка**
5. Введите username: **budgetpro_support_bot** (должен быть уникальным, добавьте цифры если занят)
6. Скопируйте токен бота — он выглядит так: `1234567890:ABCdef...`

### 13.2 — Cloudflare Worker (прокси-сервер, бесплатно)

Токен бота нельзя хранить в JS файле (его увидят все). Используем бесплатный Cloudflare Worker как посредника.

1. Перейдите на: **https://workers.cloudflare.com**
2. Создайте аккаунт (бесплатно)
3. Нажмите **Create Worker**
4. Вставьте этот код:

```javascript
const TELEGRAM_TOKEN = 'ВАШ_ТОКЕН_БОТА';
const ADMIN_CHAT_ID = 'ВАШ_TELEGRAM_ID'; // Узнайте через @userinfobot

export default {
  async fetch(request) {
    const url = new URL(request.url);
    
    // CORS заголовки
    const headers = {
      'Access-Control-Allow-Origin': '*',
      'Content-Type': 'application/json'
    };
    
    if (request.method === 'OPTIONS') {
      return new Response('', { headers });
    }
    
    if (url.pathname === '/send' && request.method === 'POST') {
      const body = await request.json();
      const text = `📱 БюджетPRO\n👤 Профиль: ${body.profile}\n💬 Сообщение: ${body.message}\n🌍 Язык: ${body.lang}`;
      
      await fetch(`https://api.telegram.org/bot${TELEGRAM_TOKEN}/sendMessage`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          chat_id: ADMIN_CHAT_ID,
          text: text,
          parse_mode: 'HTML'
        })
      });
      
      return new Response(JSON.stringify({ ok: true }), { headers });
    }
    
    return new Response('Not found', { status: 404, headers });
  }
};
```

5. Нажмите **Save and Deploy**
6. Скопируйте URL вашего воркера — выглядит как: `https://budgetpro-support.ваш-аккаунт.workers.dev`

### 13.3 — Код в index.js для отправки сообщений

```javascript
// URL вашего Cloudflare Worker
const SUPPORT_WORKER_URL = 'https://budgetpro-support.ваш-аккаунт.workers.dev';

async function sendSupportMessage(message) {
  if (!message.trim()) return;
  
  const activeProf = profiles.find(p => p.id === activeProfileId) || { name: 'Гость' };
  
  try {
    const response = await fetch(SUPPORT_WORKER_URL + '/send', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        profile: activeProf.name,
        message: message,
        lang: currentLang,
        version: '8.0',
        platform: window.Capacitor ? 'Android' : 'Web'
      })
    });
    
    if (response.ok) {
      showToast({ru:'Сообщение отправлено! Ответим в течение 24ч',en:'Message sent! We\'ll reply within 24h',ka:'გაიგ. 24სთ-ში'}[currentLang], 'success');
    }
  } catch(e) {
    showToast({ru:'Ошибка отправки. Проверьте интернет.',en:'Send error. Check internet.',ka:'შეც. ინტ.'}[currentLang], 'error');
  }
}
```

# ШАГ 14 — ДОПОЛНИТЕЛЬНЫЕ ПРОФИЛИ В НАТИВНОМ ПРИЛОЖЕНИИ

Сейчас профили хранятся в localStorage. В нативном приложении добавим постоянное хранилище.

```bash
cd ~/BudgetPRO
npm install @capacitor/filesystem @capacitor/preferences
npx cap sync
```

Добавьте в index.js:

```javascript
// Постоянное хранение профилей в нативном приложении
async function saveProfilesNative() {
  if (!window.Capacitor) return; // Только для нативного
  
  try {
    const { Filesystem, Directory } = await import('@capacitor/filesystem');
    await Filesystem.writeFile({
      path: 'budgetpro_profiles.json',
      data: JSON.stringify(profiles),
      directory: Directory.Documents,
      encoding: 'utf8'
    });
  } catch(e) {
    console.log('Filesystem save error:', e);
  }
}

async function loadProfilesNative() {
  if (!window.Capacitor) return null;
  
  try {
    const { Filesystem, Directory } = await import('@capacitor/filesystem');
    const result = await Filesystem.readFile({
      path: 'budgetpro_profiles.json',
      directory: Directory.Documents,
      encoding: 'utf8'
    });
    return JSON.parse(result.data);
  } catch(e) {
    return null; // Файл не существует — первый запуск
  }
}
```
